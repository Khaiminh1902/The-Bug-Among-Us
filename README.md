🧑‍💻 The Bug Among Us

Among Us for Coders, a fast-paced social deduction game where coding meets chaos. Work together to fix bugs before time runs out, but beware, some players are secretly sabotaging the code.

Inspired by: https://tally.so/r/aQ2kq9

🎮 Gameplay

Players are divided into two roles:

- Civilians: Collaborate to identify and fix bugs in the codebase before the timer expires
- Sabotagers: Blend in with the team while secretly introducing issues and preventing bugs from being fixed

🏆 Win Conditions

- Timer runs out and bugs remain unfixed → Sabotagers win
- Number of civilians = number of sabotagers → Sabotagers win
- All sabotagers are voted out → Civilians win

⚠️ Rules

- Communicate, but don’t fully trust anyone
- Vote wisely, one wrong decision could cost the game
- Fix bugs quickly to stay ahead of the sabotagers
